📊 Benford Pro Max | AI 自我優化版

Benford Pro Max 是一款專為審計、財務人員及數據分析師設計的輕量化、純前端在地端工具。它利用 班佛定律 (Benford's Law) 來檢測數據集是否具有人為操縱、舞弊或異常偏差的跡象。
🚀 核心功能

    多格式支援：支援上傳 .xlsx、.xls 與 .csv 檔案。

    靈活分析維度：提供「第一位數」與「第二位數」的分佈分析。

    自動化風險評估：透過 MAD (Mean Absolute Deviation) 偏差值自動判斷數據風險等級。

    視覺化圖表：即時生成實際分佈與理論分佈的對比條形圖。

    異常追蹤：自動標記疑似操縱的異常數據，並提供明細預覽。

    完整報告導出：可導出包含摘要、圖表數據、原始資料、異常與正常資料的五合一 Excel 報告。

🛠️ 技術特色

    在地端運行：所有運算均在瀏覽器內完成，資料不會上傳至任何伺服器，確保企業隱私安全。

    效能優化：系統會根據您的 CPU 核心數自動調整建議的資料處理上限（預設約每核心 10,000 筆）。

    無須安裝：單一 HTML 檔案即可運行，內建 Chart.js 與 SheetJS 函式庫。

📖 使用步驟

    上傳檔案：點擊左側上傳區，選擇您的數據檔案。

    選擇範圍：

        若 Excel 有多個工作表，請先選擇要分析的工作表。

        點擊欲分析的數字型欄位。

    設定維度：選擇要分析「第一位數」或「第二位數」。

    讀取結果：

        低風險 (MAD < 0.015)：數據符合自然分佈規律。

        高風險 (MAD ≥ 0.015)：數據可能存在人為操縱或錯誤，需進一步查核。

    導出報告：點擊「導出分析結果」獲取詳細的 Excel 稽核報告。

⚠️ 注意事項

    數據限制：建議單次處理量不超過系統提示的上限，以避免瀏覽器卡頓。

    適用範圍：最適用於正數數據。負數或包含過多小數位的數據可能會影響分析準確性。

    異常標色：系統會自動篩選前 100 筆高度偏離理論值的資料供快速檢視。
